# Boxer UIOP README

Why is this copy of uiop here?

For some of the ECL compilation options, we need to pull in a local uiop
copy for building the project. Currently, we're including this copy since
it's license compatible and it works. Of course asdf/uiop don't change too
much, but just for sanitys sake.

Hopefully we can remove this at some point.
